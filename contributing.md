# Contributing (upstream Directus v11 base)

This document reflects the contributing guidelines that applied to the upstream Directus codebase at the v11.17.4
base that d11f forked from. It is kept for historical reference only and does not describe how to contribute to d11f
itself.

Please see [https://docs.directus.io/contributing/introduction](https://docs.directus.io/contributing/introduction)
for the original upstream process as it existed at that time.

For contributing to d11f (changes made after the v11.17.4 fork point), see
[contributing-d11f.md](./contributing-d11f.md).
